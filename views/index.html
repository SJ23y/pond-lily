Welcome to this app.

Try https://pond-lily.glitch.me/wiki to use Wiki article searcher.

Try https://pond-lily.glitch.me/image to use Image Finder.

  • In the image finder you can paginate through the responses by changing offset query parameter of the URL or just press next button
  • This app use free google custom searcher api, so only 10 pages(100 images) are available in the one request
