URL Shortener Microservice

Oh hi,

Welcome to this useless app:

You can pass a URL as a parameter and You will receive a shortened URL in the JSON response.

Like that:

  • Request: https://observant-carrot.glitch.me/new/https://stackoverflow.com
  • Response: {"original_url":"https://stackoverflow.com","short_url":"https://observant-carrot.glitch.me/7"}

When You visit that shortened URL, it will redirect you to original link.

And don't ask me, why short_url longer then orignal_url. It's a miracle...
